# PNG_to_SVG

Another experience with SVG.</br>
In this algorithm the PNG file is loaded, processed and exported SVG for use in editors.</br>

## Test

- Linux
- Debian 11

## Requirements

- zlib1g
- gcc 10.2.1
- cmake 3.18.4
